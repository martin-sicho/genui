fit(X: pandas.core.frame.DataFrame, y: pandas.core.series.Series)[source]¶

This method takes the data matrix and fits the model. The input will be a DataFrame and Series. Data will usually be raw without any transformations or normalizations applied so you might want to do them here as well.

Parameters

  • X (DataFrame) – The data matrix to fit by the model. Samples as rows, variables as columns.

  • y (Series) – The ground truth value for each sample. Should be the same length as rows of X.

property model¶

You define this property so that it returns the final fitted model. It can be any object so it is ok if we just return the SVC/SVR instance directly.

This object is used mainly for serialization to disk and you can implement methods that do the job. GenUI uses joblib by default, which can handle scikit-learn instances just fine so there is no need to customize anything here.

Returns

An instance representing the fitted model.

Return type

object

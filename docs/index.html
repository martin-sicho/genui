Welcome to GenUI’s documentation!¶

This is the documentation of the GenUI backend framework. GenUI is a collection of web applications implemented with Django and Django REST. This documentation describes the Python API of the genui package, which is open source and [available on GitHub](). However, we also provide a frontend GUI application available from [here](https://github.com/martin-sicho/genui-gui). You can deploy frontend, backend and background workers easily with Docker images and Docker Compose (see Deploying a Docker Image).

Note

Currently we are using Django 2.2 as it was the most stable version at the time of starting this project, but we will begin the move to Django 3.x soon.
